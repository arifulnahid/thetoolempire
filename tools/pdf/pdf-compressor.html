How to Compress a PDF

1
Drop your PDF or click Choose PDF
Open the PDF you want to compress. The tool loads it into your browser and shows the file name, size, and page count.
2
Select a compression preset
Choose from four presets. Medium is recommended for most files — it significantly reduces size while keeping content readable. Use Low for the smallest possible output.
3
Click Compress PDF
The tool processes every page, compressing each one to a JPEG at the chosen quality level and rebuilding a new PDF. A progress bar shows how many pages have been processed.
4
Review and download
The result shows original vs compressed sizes and savings percentage. If the result isn't small enough, click "Try a different preset" and compress again with a lower quality level.

About This Tool

PDF Compressor reduces PDF file size by compressing embedded images and removing redundant metadata, optimising the file for email attachments or web uploads — without re-encoding text or vector graphics.

🛠

How to Use

  1. Upload a PDF file.
  2. Choose compression level: Low (best quality), Medium, or High (smallest size).
  3. Click Compress.
  4. View the before/after file size comparison and download.
⚙️

Logic & Algorithm

PDF images are extracted by identifying image XObjects in the PDF content streams using pdf-lib. Each image is decoded, drawn onto a <canvas> at reduced quality using toDataURL('image/jpeg', quality), re-encoded, and replaced in the PDF XObject dictionary. Metadata streams are removed by deleting optional XMP entries.
