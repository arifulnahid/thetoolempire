Frequently Asked Questions

What chart types are available?
9 types are supported: Bar (vertical), Horizontal Bar, Line, Area (filled line), Pie, Doughnut (ring), Radar (spider), Polar Area, and Scatter. Each supports multiple datasets.
How do I add multiple datasets to a bar or line chart?
Open the Data tab and click "Add Dataset". Each dataset gets its own label, values, and color. For bar/line charts they render as grouped series. Enable "Stacked" in the Style tab to stack them instead.
How do I enter scatter plot data?
Select "Scatter" chart type, then in the dataset values field enter one x,y pair per line (or separate pairs with semicolons). For example: 1.5, 2.8 on the first line, 3.2, 5.1 on the next.
What download formats are supported?
PNG (lossless, great for web and presentations), JPEG (smaller file, slight quality loss), SVG (vector format, scalable to any size), and PDF (A4 page, landscape or portrait automatically chosen). All downloads include a dark background matching the tool.
Can I pick my own colors?
Yes. In the Data tab, click the colored dot next to any dataset name to open a color picker. The palette presets in the Style tab set the default starting colors when you add new datasets, but per-dataset colors always take precedence.
