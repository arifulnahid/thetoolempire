How to Combine Images into a PDF

1
Add your images
Drag images onto the upload area or click Choose Images. Add as many images as you need — each one becomes a page in the PDF. You can add more at any time using the Add Images button.
2
Reorder pages (optional)
Use the ↑ and ↓ buttons next to each image to move it up or down in the page order. The page number badge on the left updates as you reorder.
3
Configure PDF settings
Choose page size (A4, Letter, etc.), orientation (portrait or landscape), how the image fits the page, and the margin. The default settings work well for most use cases.
4
Click Generate PDF
The PDF is built in your browser and downloaded automatically. Check the filename field if you want to rename it before saving.

About This Tool

Image to PDF Converter embeds JPEG, PNG, WebP, and GIF images into a PDF document — one image per page or multiple per page with layout options. Supports batch uploads and custom page sizes.

🛠

How to Use

  1. Upload one or more images.
  2. Choose page size (A4, Letter, custom) and orientation.
  3. Arrange images by dragging to reorder.
  4. Select fit mode: fill page, fit within margins, or original size.
  5. Click Convert and download the PDF.
⚙️

Logic & Algorithm

Uses pdf-lib: a new PDFDocument is created, then each image is embedded with embedJpg() or embedPng(). A new page is added for each image (or images are tiled using layout math). The image is drawn with page.drawImage(img, { x, y, width, height }) after scaling to fit the page with aspect-ratio preservation.
